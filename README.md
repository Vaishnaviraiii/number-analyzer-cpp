# number-analyzer-cpp
Beginner-friendly C++ project demonstrating functions, loops, and menu-driven programming.
## Features
- Check if a number is prime
- Find sum of digits
- Reverse a number
- Check if a number is palindrome
- Count number of digits
- Option to change input number
## Concepts Used
- Functions
- Loops(do-while, for)
- Conditional statements (if-else, switch)
- Menu-driven programming
## How to run
1. Open terminal
2. Compile the code:
   g++ numberAnalyzer.cpp
3. Run the program:
   ./a.exe
## File
- numberAnalyzer.cpp -> main source code
## About
This project was built as part of learning C++ fundamentals and practicing functions and logic building.
